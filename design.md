# htxswim Landing Page - Design Direction

## Brand
- Artist: htxswim (YouTube: swimmyxo)
- Genre: Rap & Rock
- Vibe: Dark, aggressive, raw energy, streetwear aesthetic

## Colors
- Background: Pure black (#000000)
- Primary accent: Blood red (#DC2626 / #EF4444)
- Secondary: Dark grays (#111111, #1a1a1a)
- Text: White (#FFFFFF) and muted gray (#888888)
- Glow effects: Red with opacity

## Typography
- Display: "Bebas Neue" — bold, condensed, impactful for headings
- Body: "Space Mono" — techy monospace for body text
- Style: ALL CAPS for headings, mixed for body

## Layout
- Single page, full-bleed sections
- Hero: Full viewport with 3D animated element + artist name
- Below: Social/streaming links grid
- Minimal navigation — it's a link-in-bio style page

## 3D Element
- Rotating geometric shape (icosahedron wireframe) with red glow
- Floating particles in background
- Subtle rotation on mouse movement

## Motion
- Staggered entrance animations on load
- Smooth scroll between sections
- 3D element continuous rotation
- Hover effects on link cards

## Anti-patterns to avoid
- No rounded friendly cards
- No light mode
- No generic SaaS look
- No purple gradients
